### ⚠️ Warning Message

NappyCraft is a mod that draws inspiration from the Regression and Disable communities. For the regression/roleplay side, we follow the principles of Regressia — a community built around friendly, wholesome, and 100% clean regression roleplay.

Within this philosophy, nappies/diapers are treated in the same way as underwear: a private item that should not be flaunted publicly. The creator of NappyCraft fully supports this idea. While Minecraft servers require users to manually join before seeing any in-game content, we still feel it is important to provide a clear notice to new players:
➡️ Some servers using this mod may display nappies/diapers as part of the experience.

NappyCraft does not condone the misuse of regression themes to harass, embarrass, or abuse other players. Our goal is to keep this mod fun, safe, and PG-13/SFW for everyone.

If you ever encounter a server that uses NappyCraft in an unsafe, non-PG-13 way, and they are promoting our mod, please let us know so we can take appropriate action.

🔔 This warning message should only appear once per player in-game.

🔔 By default, this warning message will appear once per player in-game.
🌐 For LAN or multiplayer server hosts: please ensure your players are made aware that NappyCraft is installed, as the mod introduces regression-themed content.
⚙️ Server owners can control this via the doNappyWarning gamerule:

 * true (default) → new users see this warning once.
 * false → disables the warning for new users.
